# Examples of Scala Cite Services Requests

> Assuming the server is running at, *e.g.* `localhost:9000`

## Texts

### See the Text Catalog

- <http://localhost:9000/texts>

### Get Valid References

- <http://localhost:9000/reff/urn:cts:greekLit:tlg0016.tlg001.grc:>
- <http://localhost:9000/reff/urn:cts:greekLit:tlg0016.tlg001.grc:2>
- <http://localhost:9000/reff/urn:cts:greekLit:tlg0016.tlg001.grc:2-3>
- <http://localhost:9000/reff/urn:cts:greekLit:tlg0012.tlg001:1>
